# Projet ML - Prédiction d'Attrition Employé

Ce projet permet de prédire le risque d'attrition d'un employé à partir de données RH.

## Contenu du projet

- API FastAPI
- Interface Gradio
- Modèle de Machine Learning
- Base PostgreSQL
- Traçabilité des échanges API ↔ BDD
- Tests Pytest
- Rapport de couverture Pytest-cov
- Documentation MkDocs

## Objectif

L'objectif est de fournir une solution complète permettant :

1. d'envoyer des données employé
2. d'obtenir une prédiction
3. d'enregistrer les inputs et outputs
4. de tracer les échanges
5. de déployer le projet sur GitHub et Hugging Face
